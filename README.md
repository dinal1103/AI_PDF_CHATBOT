## Overview

DocuMind AI is a Retrieval-Augmented Generation (RAG) system built to demonstrate core NLP/ML engineering skills: dense embeddings, vector similarity search, retrieval re-ranking, and LLM-grounded question answering — with a production-style FastAPI backend and Streamlit frontend.
